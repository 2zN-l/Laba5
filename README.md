# Лабораторная работа №5

### Генератор, который обходит файловую систему в указанном каталоге и возвращает имена файлов с заданным расширением. Предусмотреть возможность обхода подкаталогов. Переименуйте найденные файлы.

Проделанная работа:
1. Поиск всех файлов с заданным расширением в указанной директории с возможностью рекурсивного обхода папок;
2. Фильтрация файлов по расширению без учёта регистра;
3. Генерация последовательности файлов для переименования;
4. Перебор файлов с нумерацией, начиная с 1;
5. Формирование нового имени файла;
6. Переименование каждого файла с заменой старого пути на новый;
